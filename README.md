### Topic: Earthquake-Vulnerable Buildings in Seattle
I've been curious about this dataset I've found on the Seattle Open Data Portal for a while, since we have a lot of earthquakes in the PNW, so I figured I would map it to see what the distribution of these buildings is like. It shows the location of all the registered Unreinforced Masonry Buildings in the city that might collapse in the event of a major earthquake in the region. I decided to make a proportional symbols map, as I wanted more specific location of where the buildings were located than a choropleth map would provide, since I'm familiar with the area, and also the symbol size and colors could be used to emphasize the level of risk each building has according to the data I found. The example also used this type of map, which made it easier for me to figure out how to get the html working to display what I wanted it to.
