# Earthquake-vulnerable buildings in the displaying region

Seattle is built in a seismically active area, and contains numerous buildings constructed before the advent of modern structure codes and safety requirements. This map displays city-provided data on where these buildings are located, and a vulerability index determined by a few different factors. Below is the link to the description of the process for determining the risk category, which is also given in the source on the map legend: https://data-seattlecitygis.opendata.arcgis.com/datasets/SeattleCityGIS::unreinforced-masonry-buildings-urm/about
